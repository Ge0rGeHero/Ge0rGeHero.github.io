# 2024-10-20
读到102页，开始接受**亚里士多德**的哲学思想，然后对于亚里士多德和他老师**柏拉图**的区别，我觉得，柏拉图说可以从事物的特殊性中，找到相的普遍性，但他偏向否定人的感官获得的知识，因为事物只是相的一部分，所以人难以得知相的全部，通过感官获得的信息意义并不大，而且柏拉图提供找到相的方法并不完善。  
而亚里士多德，他肯定了感官获得的信息，并提出了归纳法，让人们可以通过感觉*感觉经验*推导出真理。  
不得不说，还是好感慨，现在的逻辑学，竟然是这么多年前的人（亚里士多德）发现的，我还以为这是比较近代才发现的东西，太厉害了。柏拉图和亚里士多德这两者的哲学思想，我应该更偏好亚里士多德的吧。
# 2024-10-07
## 关于苏格拉底“善”的观点的看法
苏格拉底认为，人只有在认识真正的“善”就会行善，有些人明知偷钱是不道德的，但还是会偷，那是因为他被欲望给蒙蔽了，偷钱时他并没有看到真正的“善”。我觉得这有一定的道理，我知道做坏事是不行的，但我在做坏事时被蒙蔽了，所以那时也算是我不知道真正的“善”。  
但是，苏格拉底并没有对人为什么会在知道（被教育）真正的善后，还是会被欲望（或其他东西）蒙蔽双眼做出解释，一个人，他心地善良，但是他为了生活，只能迫不得已去做坏事。苏格拉底的观点没有解释人为什么会被蒙蔽，以及如何解决被蒙蔽的问题，这是苏格拉底观点的一个缺陷。
## 关于洞穴寓言的看法
洞穴寓言一定程度上映射了苏格拉底的处境，看到真相的人回到洞穴后，告诉洞穴的人真相，但被当作疯子处死了。  
但我在想，如果回到洞穴的那个人，他不是靠嘴巴说出真相，而是把那面遮住真相的墙破坏掉，让真理的光芒照进其他的人内心，那他们自然就会看到真相是什么，那也不会把他当成疯子了。所以这也说明了只有采取暴力手段，才会真相大白，真理应当属于所有人的。
# 2024-10-05
目前看到80页，对**苏格拉底**和**柏拉图**的哲学观点有了一定的了解，柏拉图的观点基本上是师承苏格拉底并对其进行一定改进的，其中柏拉图的知识论让我感触很深，这完全可以运用到我们的日常生活的。  
柏拉图对知识论的看法：首先，真理是实在的，然后**相**是真理的对象，相的一部分会如影子一般映射到物理世界，有点像《三体》中对质子的描述一样，我们对四维世界的质子永远只能看到一部分。但是我们通过**理性**地推理，明确地知道它是确实存在的，我们通过观察归纳它的部分，从而推导出它的基本属性。

对“数学的终点是哲学”的反驳：会有这种观点的人明显就是不懂哲学、数学、物理，就如上面柏拉图对知识论的看法，我们人类所要追求的就是那个抽象的、实在的终极真理，通过无需假设的基本原则来推导出其他原则，那么，物理其实探索的就是从自然事物中，总结它们的基本属性。如：根据光的衍射等光的**相**在物理世界中的映射推导出光的传播公式。同理，数学就是从“1+1=2”“三角定理”等基本原则中推导出其他定理，哲学就是从人类的日常经验中推导出“正义”“公平”等本质。可见，哲学、物理、数学等其实是对**相**探索的不同方向而已，完全没有“数学的终点就是哲学”一说。
# 2024-09-28
目前看到第58页，真的很喜欢这本书，非常能激发我的思考。不得不感慨，以前高中、初中学的政治都是些什么玩意儿，全都是死记硬背，从未讲解过这些哲学家背后的故事以及思想，就只会拿个例子然后就让别人猜答案，比如：
```
泰勒斯踩到水坑，说明天会下雨，然后问这表明了什么思想。
答：元素学说
```
这背后明明是有背景的，是因为泰勒斯觉得世界是有单一元素构成的，以前课上却从未提过这点。  
然后看到目前的进度，不得不感慨古人的想象力，比如那个原子论，和现在科学发现的事实非常的接近，虽然以前的原子论并没有什么具体推理和数据支撑，这是不是某种程度称得上是最早期的科幻了。而以前的哲学家的思想，有些问题是现在还在探讨的，总之就是很感慨，很喜欢这本书。
